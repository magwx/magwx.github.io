# What is this?

Code for [my personal website](https://magwx.github.io) built with [Jekyll](http://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## Requirements

```sh
gem install github pages
```

## Usage

```sh
jekyll serve
```
